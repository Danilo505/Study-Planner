# Study Planner

Aplicação full stack para planejamento inteligente de estudos.

## Objetivo

Ajudar estudantes a organizar estudos automaticamente com base em:
- prazo de provas
- dificuldade do conteúdo
- tempo disponível por dia

## Stack

Backend:
- Java 21
- Spring Boot
- Spring Security
- PostgreSQL
- Flyway

Frontend:
- React
- TypeScript
